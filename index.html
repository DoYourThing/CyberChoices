Keyboard commands:
'right arrow' = next
'left arrow' = back
'r' = replay current text bubble
'esc' = home
't' = try again
'Enter' = accept
'h' = help
'o' = options
'x' = cancel (close popups)
'1' = sort reason left
'2' = sort reason right
'1' = select action 1 when choices are unlocked
'2' = select action 2 when choices are unlocked
